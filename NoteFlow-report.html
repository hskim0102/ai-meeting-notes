우리 조직의 회의, 이대로 괜찮을까요?

반복되는 비효율 — 기록되지 않는 결정, 추적되지 않는 액션 아이템

✍️

회의록 작성 부담

담당자가 회의에 집중하면서 동시에 기록하는 것은 현실적으로 불가능합니다. 회의가 끝난 후 기억에 의존해 작성하면 발언 내용과 결정 사항이 누락되기 쉽습니다.

❌

액션 아이템 미이행

구두로 합의한 사항은 문서로 남지 않아 담당자, 기한, 내용이 불분명해집니다. 누가 무엇을 언제까지 해야 하는지 명확히 추적되지 않아 실행력이 떨어집니다.

🔍

지식 단절

이전 회의에서 논의된 내용을 다시 찾기 위해 메신저, 이메일, 파일을 뒤지는 데 많은 시간이 낭비됩니다. 담당자 퇴직 시 조직 지식이 함께 소실됩니다.

🔒

정보보안 리스크

회의 중 언급된 개인정보·민감 정보가 그대로 문서화되어 유출 위험이 발생합니다. 접근 제어 없이 모든 회의록이 공유되는 구조도 문제입니다.

⏱

시간 낭비 — 회의록 1건당 평균 30~60분 추가 소요

주 5회 회의 기준, 1인당 연간 최대 200시간을 회의록 작성에 낭비합니다. 이는 약 25일치 근무 시간에 해당합니다.

💡

NoteFlow가 이 모든 문제를 해결합니다

녹음 버튼 하나로 STT 전사 → AI 요약 → 액션 아이템 추출 → 지식 축적까지 자동화. 회의가 끝나는 순간 완성된 회의록이 준비됩니다.

기능 1 — 음성 녹음 & AI 음성 인식 (STT)

회의를 기록하는 가장 자연스러운 방법 — 그냥 말하면 됩니다

🎤

실시간 녹음

  • 브라우저 내장 마이크로 즉시 녹음 시작
  • 녹음 중 실시간 자막 표시 (Web Speech API)
  • WebM 포맷 자동 저장, 서버 보관
  • 녹음 중단 후 STT 자동 실행
📂

파일 업로드

  • MP3, MP4, WAV, WebM 등 다양한 포맷 지원
  • 드래그앤드롭 업로드
  • 최대 500MB — 장시간 회의도 처리 가능
  • 25MB 초과 파일 자동 분할 & 순차 전사
🧠

AI 음성 인식 (Whisper)

  • OpenAI Whisper — 세계 최고 수준의 STT 정확도
  • 한국어 비즈니스 대화 최적화
  • 무음 구간 환각(hallucination) 자동 필터링
  • 화자 분리(Diarization) 선택 적용
✏️

키워드 교정

  • 잘못 전사된 업무 용어·인명·프로젝트명 교정
  • AI 키워드 자동 추출 → 사용자 확인·수정
  • 교정 적용 시 전체 텍스트 일괄 치환
  • 화자별 발언 타임라인 인라인 수정
지원 포맷MP3, MP4, WAV, WebM, M4A
최대 파일 크기500 MB
자동 분할 기준25 MB 초과 시
STT 언어한국어 (기본), 다국어 지원
