## 1. 도입 배경 및 목적

### 현재의 문제

조직 내 회의는 매일 반복되지만, 회의에서 논의된 내용의 **대부분이 기록되지 않거나, 기록되더라도 활용되지 않습니다.**

| 문제 | 현황 |
|------|------|
| 회의록 작성 부담 | 담당자가 회의에 집중하면서 동시에 기록하는 것은 현실적으로 어려움 |
| 사후 기록의 부정확성 | 회의 후 기억에 의존하여 작성 → 발언 내용·결정 사항 누락 빈번 |
| 액션 아이템 미이행 | 누가, 무엇을, 언제까지 해야 하는지 명확히 추적되지 않음 |
| 지식 단절 | 이전 회의에서 논의된 내용을 찾기 위해 불필요한 시간 낭비 |
| 보안 우려 | 회의 중 언급된 개인정보·민감 정보가 그대로 문서화됨 |

### NoteFlow의 목적

**NoteFlow**는 이 문제를 AI 기술로 해결합니다.

- **녹음만 하면** STT(음성-텍스트 변환)로 전사
- **전사 텍스트를** AI가 대기업 부서장급 수준의 공식 회의록으로 요약
- **결정 사항과 액션 아이템**을 자동 추출하여 추적
- **모든 회의록이 지식으로 축적**되어 언제든 검색·질의 가능
- **개인정보를 자동 마스킹**하여 정보보안 리스크 최소화

## 2. 주요 기능

### 2-1. 음성 녹음 및 파일 업로드

| 방식 | 설명 |
|------|------|
| **실시간 녹음** | 브라우저에서 바로 마이크 녹음. 녹음 중 실시간 자막 표시 |
| **파일 업로드** | 기존 녹음 파일(MP3, MP4, WAV, WebM 등) 드래그앤드롭 업로드 (최대 500MB) |
| **긴 녹음 자동 처리** | 25MB 초과 파일은 자동으로 분할 후 순차 전사·병합 |

---

### 2-2. AI 음성-텍스트 변환 (STT)

OpenAI Whisper 기반의 고정밀 음성 인식을 제공합니다.

- **한국어 최적화** — 한국어 회의 대화 전사에 최적화
- **자동 환각 필터링** — 무음 구간의 AI 오류 발언(환각)을 자동 감지·제거
- **화자 분리** — 여러 명이 참여한 회의에서 발화자별로 텍스트를 분리
- **키워드 교정** — 잘못 전사된 업무 용어, 인명, 프로젝트명을 사용자가 일괄 교정

---

### 2-3. AI 회의록 자동 생성

전사된 텍스트를 Dify AI 워크플로우가 분석하여 **공식 회의록 수준의 구조화된 문서**를 즉시 생성합니다.

#### 생성되는 항목

```
① 회의 요약 (Summary)
   - 회의 배경 및 목적
   - 주요 논의 항목별 상세 내용 (찬반 논거 포함)
   - 핵심 의사결정 및 결정 근거
   - 우려사항 및 리스크
   - 결론 및 향후 방향

② 핵심 결정사항 (Key Decisions)
   - "무엇을 → 어떻게 → 왜" 구조로 정리

③ 액션 아이템 (Action Items)
   - 수행 과제 / 담당자 / 마감일 자동 추출

④ 핵심 키워드 / 태그
   - 프로젝트명, 기술명, 핵심 개념어 자동 분류

⑤ 회의 분위기 분석 (Sentiment)
   - 긍정적 / 부정적 / 중립적 자동 판정
```

#### 회의록 품질 기준

- 300자 이상의 상세 요약 (단순 나열 아닌 맥락-논거-결론 서술)
- 사실에 근거한 내용만 작성 (스크립트에 없는 내용 추가 금지)
- 발화자 인용 포함 ("OOO님 — '핵심 발언 인용'")

---

### 2-4. 액션 아이템 통합 관리

회의에서 추출된 할 일 목록을 체계적으로 추적합니다.

| 기능 | 내용 |
|------|------|
| **자동 추출** | "~하기로 함", "OOO이 ~함" 등의 발언에서 자동 추출 |
| **담당자·마감일** | 회의 발언에서 자동 인식하여 기입 |
| **상태 관리** | 대기 / 진행 중 / 검토 / 완료 4단계 추적 |
| **전사 뷰** | 모든 회의의 액션 아이템을 한 페이지에서 통합 조회 |
| **우선순위 필터** | 담당자·마감일·우선순위별 필터링 |

---

### 2-5. AI 챗봇 (회의록 기반 Q&A)

축적된 회의록 데이터를 기반으로 자연어로 질의할 수 있습니다.

```
예시 질문:
- "지난달 예산 회의에서 마케팅 관련 결정된 게 뭐야?"
- "3분기 제품 출시 일정이 어떻게 됐어?"
- "OOO 프로젝트의 리스크로 뭐가 언급됐었지?"
```

| 모드 | 설명 |
|------|------|
| **단일 회의 Q&A** | 특정 회의에 대한 질문·답변 (이전 대화 맥락 유지) |
| **전체 회의 검색 Q&A** | 모든 회의에서 관련 내용을 검색해 답변 |

---

### 2-6. 통합 검색

가중치 기반 스코어링으로 관련성 높은 회의록을 정밀 탐색합니다.

| 검색 대상 | 가중치 |
|-----------|--------|
| 회의 제목 | 3.0 (최우선) |
| 태그·키워드 | 2.5 |
| AI 요약 | 2.0 |
| 액션 아이템 | 1.5 |
| 참석자 | 1.5 |
| 전사 전문 | 1.0 |

- **고급 필터** — 날짜 범위, 참석자, 태그, 회의 분위기별 필터
- **자동완성** — 참석자명, 태그 입력 시 자동 제안

---

### 2-7. 대시보드 및 분석

- **통계 현황** — 전체 회의 수, 액션 아이템 완료율, 이번 달 회의 추이
- **미완료 액션 아이템** — 대시보드에서 즉시 확인
- **회의 트렌드** — 기간별 회의 분포, 참석자별 통계, 키워드 트렌드

---

### 2-8. 회의록 메일 발송

- 회의 종료 후 참석자에게 회의록을 즉시 이메일 발송
- 개인정보 마스킹된 버전 선택 발송 가능
- 참석자 외 추가 수신자 지정 가능

---

### 2-9. 회의실 예약 시스템

- 건물·수용 인원별 회의실 필터 조회
- 주간 캘린더 뷰로 실시간 가용성 확인
- 예약 생성·수정·취소 관리

---

## 3. 편의 기능

### 3-1. 개인정보 자동 마스킹

회의 저장 시 AI가 자동으로 개인정보를 감지하여 비식별화합니다.

- 주민번호, 연락처, 계좌번호 등 민감 정보 자동 마스킹
- 원본과 마스킹 버전을 별도 저장 (용도에 따라 선택 활용)
- 이메일 발송 시 마스킹 버전 선택 가능 → 정보보안 컴플라이언스 지원

---

### 3-2. 권한 기반 접근 제어

| 역할 | 권한 |
|------|------|
| **관리자 (Admin)** | 전체 회의록 조회·수정·삭제, 사용자 관리, 감사 로그 조회 |
| **매니저 (Manager)** | 자신과 팀 회의록 조회·수정 |
| **멤버 (Member)** | 자신이 참석한 회의록만 조회 |

---

### 3-3. 실시간 자막

녹음 중 Web Speech API 기반으로 발화 내용이 화면에 실시간 표시됩니다.

- 회의 진행 중 즉각적인 텍스트 확인 가능
- 청각 장애인 또는 소음 환경에서도 내용 파악 가능

---

### 3-4. 커맨드 팔레트 (Ctrl+K)

키보드 단축키로 앱 전체를 빠르게 탐색합니다.

- 페이지 이동: 회의 목록, 새 회의, 검색, 대시보드 즉시 이동
- 회의 검색: 입력 즉시 관련 회의 탐색
- 마우스 없이 전체 기능 접근 가능

---

### 3-5. 다크 모드

- 시스템 설정 자동 감지
- 수동 전환 및 설정 저장
- 눈의 피로 감소, 야간 작업 환경 최적화

---

### 3-6. 반응형 UI

- **PC** — 사이드바 + 넓은 콘텐츠 영역
- **태블릿** — 접이식 사이드바
- **모바일** — 하단 탭 네비게이션

---

### 3-7. 감사 로그

- 모든 회의록 생성·수정·삭제 이력 기록
- 관리자가 언제, 누가, 무엇을 변경했는지 추적 가능
- 내부 컴플라이언스·감사 대응 지원

---

### 3-8. 녹음 파일 보관

- 업로드하거나 녹음한 오디오 파일을 서버에 보관
- 보관된 녹음에서 STT 재실행 가능
- 녹음과 회의록을 연결하여 원본 음성 재청취 가능

---

## 4. 사용 시 기대 효과

### 4-1. 시간 절감

```
기존 방식:
  회의 중 기록 + 회의 후 정리 = 평균 30~60분 추가 소요

NoteFlow 도입 후:
  녹음 시작 → AI 요약 → 검토·보완 = 5~10분
  
  → 회의록 1건당 약 20~50분 절감
  → 주 5회 회의 기준 연간 약 80~200시간 절감 (1인 기준)
```

---

### 4-2. 회의 집중도 향상

- 기록 걱정 없이 토론에 집중 가능
- 발언 인용, 논거 포착을 AI가 대신 처리
- 회의 품질과 의사결정 속도 향상

---

### 4-3. 조직 지식 자산화

- 모든 회의록이 검색 가능한 지식 베이스로 축적
- "예전에 그 안건 어떻게 결정했더라?" → 챗봇에 바로 질문
- 신입 구성원의 업무 온보딩 속도 향상 (과거 회의 맥락 파악 용이)
- 퇴사·이직에도 조직 지식 유실 방지

---

### 4-4. 책임 명확화 및 이행률 향상

- 액션 아이템의 담당자·마감일 자동 추출 → 구두 약속이 문서로 전환
- 전체 액션 아이템 현황을 한 눈에 파악
- 마감 임박 항목 우선 표시 → 이행률 향상

---

### 4-5. 정보보안 강화

- 회의 중 언급된 개인정보 자동 마스킹
- 역할 기반 접근 제어 → 필요한 사람만 필요한 회의록 접근
- 모든 접근·변경 이력 감사 로그 보존

---

### 4-6. 일관된 회의록 품질

- AI가 항상 동일한 양식·수준으로 요약 → 작성자에 따른 품질 편차 제거
- 공식 회의록 템플릿(개요·안건·논의·결정·액션·리스크) 자동 적용

---

## 5. 시스템 구성

### 5-1. 기술 스택

| 구분 | 기술 |
|------|------|
| **프론트엔드** | Vue.js 3, Tailwind CSS, Chart.js |
| **백엔드** | Node.js, Express.js |
| **데이터베이스** | MySQL (FULLTEXT 검색 인덱스 활용) |
| **AI 요약·챗봇** | Dify AI Workflow / Chat API |
| **음성 인식** | OpenAI Whisper API |
| **화자 분리** | Pyannote (선택 적용) |
| **메일 발송** | Nodemailer (SMTP) |
| **배포** | Docker, Nginx |

---

### 5-2. 데이터 흐름

```
사용자 (녹음/업로드)
        ↓
  STT 변환 (Whisper)
        ↓
  키워드 교정 (선택)
        ↓
  AI 요약 생성 (Dify Workflow)
  ┌─ 회의 요약
  ├─ 결정사항
  ├─ 액션 아이템
  ├─ 키워드
  └─ 회의 분위기
        ↓
  DB 저장 + RAG 지식화 (비동기)
        ↓
  개인정보 마스킹 (비동기)
        ↓
  메일 발송 (선택)
```

---

### 5-3. 저장 정보 구조

회의 1건당 저장되는 주요 정보:

| 필드 | 내용 |
|------|------|
| 제목·날짜·시간·장소 | 회의 기본 정보 |
| 참석자 목록 | 이름 배열 |
| AI 요약 | 300자 이상 상세 요약 |
| 핵심 결정사항 | 확정 결정 목록 |
| 액션 아이템 | 과제·담당자·마감일·완료 여부 |
| 키워드/태그 | 검색용 핵심어 |
| 회의 분위기 | 긍정/부정/중립 |
| 화자별 전사 | 발화자·시간·발언 구조화 데이터 |
| 전문(Full Text) | 전체 전사 텍스트 (검색용) |
| 마스킹본 | 개인정보 제거된 별도 저장본 |

---

## 6. 향후 발전 방향

### 6-1. 다국어 지원

- 영어, 일본어, 중국어 STT 및 요약 지원
- 글로벌 팀 회의, 해외 파트너 미팅 회의록 자동 번역
- 다국어 회의 내용을 한국어로 요약 (언어 장벽 제거)

---

### 6-2. 캘린더 통합 및 사전 준비 지원

- Google Calendar, Outlook 연동
- 예약된 회의에 맞춰 회의실·참석자 자동 설정
- 이전 관련 회의 요약을 회의 전 자동 공유 ("회의 준비 브리핑")
- 회의 안건 사전 공유 및 의견 수렴 기능

---

### 6-3. 실시간 다자간 회의 지원

- 화상회의(Zoom, Teams, WebEx) 연동
- 회의 중 실시간 자막·요약 표시
- 원격 참석자와 현장 참석자 음성 동시 전사
- 회의 종료 즉시 요약 자동 생성 및 공유

---

### 6-4. 액션 아이템 외부 연동

- Jira, Trello, Notion, 사내 업무 시스템과 연동
- 회의에서 추출된 액션 아이템을 클릭 한 번으로 티켓 생성
- 마감일 임박 시 Slack, 이메일, 모바일 푸시 알림
- 이행 여부를 외부 시스템과 실시간 동기화

---

### 6-5. 회의 품질 분석 및 인사이트

- 회의 분위기 트렌드 분석 (부서별, 월별)
- 발언 비중 분석 — 특정 참석자 편중 여부 시각화
- 액션 이행률 리포트 — 팀·담당자별 이행 현황
- 회의 효율 점수 — 결정사항 없이 끝난 회의 감지
- 유사 회의 감지 — 같은 주제 반복 논의 알림

---

### 6-6. 모바일 앱

- iOS / Android 네이티브 앱 출시
- 이동 중 회의 녹음 및 즉시 요약
- 오프라인 녹음 후 네트워크 복구 시 자동 업로드
- 위젯으로 금일 회의 및 액션 아이템 빠른 확인

---

### 6-7. 보안 및 컴플라이언스 강화

- 온프레미스(On-Premise) 배포 옵션 — 데이터 외부 유출 없이 사내 운영
- AI 처리 내부화 — 클라우드 API 미사용 완전 폐쇄망 지원
- 회의록 전자서명 — 결정사항에 참석자 전자 서명 첨부
- 법적 효력 보존을 위한 타임스탬프 및 불변 로그

---

### 6-8. 조직 지식 고도화

- 부서별 전문 용어 사전 구축 → STT 정확도 향상
- 회의록 기반 FAQ 자동 생성
- 온보딩 가이드 자동 생성 — 신입 사원에게 관련 과거 회의 맥락 제공
- 의사결정 이력 추적 — "이 정책이 어느 회의에서 왜 결정됐는가" 추적 가능
